# jupyterbook_Modelacion_Ambiental

Book del curso de Modelacion Ambiental construido con MyST y Jupyter Book.

## Requisitos

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Desarrollo local

```bash
./start-book.sh
```

## Build

```bash
python -m jupyter_book build .
```
